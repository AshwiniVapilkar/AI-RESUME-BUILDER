In this project, React (with Vite) is used to build a fast, interactive user interface, allowing users to create resumes smoothly. Tailwind CSS handles styling, making the layout modern and responsive. Clerk manages user authentication, enabling secure sign-up and login features. The app integrates an AI service to automatically generate resume summaries based on user input. For backend storage, Strapi CMS is used to save and manage resume data.
